# Cursor Plugin Marketplace

This repository is a [Cursor plugin marketplace](https://cursor.com/docs/reference/plugins#multi-plugin-repositories). Plugins are listed in `.cursor-plugin/marketplace.json`.

## Plugins

| Plugin | Description |
|--------|-------------|
| [po-jira](po-jira/) | Jira planning and ticket workflows (rules, skills, auditor agent, commands, MCP). |

See each plugin’s `README.md` for setup and usage.
